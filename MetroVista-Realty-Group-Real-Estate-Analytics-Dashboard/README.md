# MetroVista-Realty-Group
This project is an interactive Power BI dashboard created to analyze the sales performance of a real estate company, MetroVista Realty Group. The dashboard provides insights into property sales, revenue trends, location performance, and agent productivity to help management make data-driven decisions.

Objectives

Analyze overall real estate sales performance

Track monthly and yearly sales trends

Identify top-performing locations and property types

Evaluate sales agent performance

Monitor total revenue and number of properties sold
